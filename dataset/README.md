# Research Materials for Master's Thesis

This repository contains all research materials, datasets, survey forms, evaluation results, and supplementary data used in the master's thesis:

**“중고 건설기계 거래 플랫폼의 사용자 경험 최적화를 위한 인공지능 통합 설계 연구”**

본 저장소에는 연구 과정에서 사용된 **데이터·설문·부록 자료만 포함**되며,  
**AI 모델 소스코드 및 서비스 코드(repository code)는 포함되어 있지 않습니다.**

---

## 📁 Repository Structure

### **1. survey/**
사용자 경험 평가(SUS, UEQ, SEQ) 및 사용성 테스트 분석을 위해 사용된 설문 관련 자료.
- 설문지 양식 (survey_form.pdf)  
- 설문 문항 원본 (survey_original_questions.pdf)  
- 응답 정리 및 점수 환산 파일 (SUS/UEQ/SEQ)  
- 비식별화된 통계 결과자료  

---

### **2. dataset/**
AI 실험 및 시스템 평가에 사용된 기초 데이터셋.
- **trade_data_clean.xlsx**  
  비식별화된 중고 장비 기초 거래 데이터  
- **behavior_scenario.pdf**  
  추천 알고리즘 설계에 사용된 사용자 행동 로그 시나리오  
- **system_performance.xlsx**  
  YOLO·Multi-head 분류·시세예측·추천 시스템의 API 응답 시간 측정 결과  

---

### **3. appendix/**
논문 부록 자료 및 연구 참고 문서.
- 사용자 테스트 시나리오  
- 연구에 활용된 참고 문서  
- IRB/동의서(필요할 경우)  

---

## 🎯 Purpose of This Repository
- 논문 검증 및 재현성(Replicability)을 제공하기 위함  
- 연구에 사용된 설문·데이터셋·실험 결과를 투명하게 공개  
- 외부 연구자 및 심사위원이 실험 환경을 이해할 수 있도록 구성
