# Research Materials for Master's Thesis

This repository contains all research materials, datasets, survey forms, evaluation results, and supplementary data used in the master's thesis:

**"모바일 기반 중고 건설장비 거래 플랫폼의 사용자 경험 최적화를 위한 인공지능 통합 설계 연구"**

Only research materials are included. No source code is stored in this repository.

## Structure
- survey/: 설문 원본, SUS·UEQ·SEQ 결과
- dataset/: 거래 데이터, 행동 로그, 잔가율표
- appendix/: 실험 시나리오, 사용자 테스트 문서
- evaluation/: 모델 성능 및 시스템 평가 결과
